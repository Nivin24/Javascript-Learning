# Arrays and Objects Quiz

### Q1: How do you access the third element of an array?
**Answer:** array[2] (index starts at 0)

### Q2: What method adds an element to the end of an array?
**Answer:** push()

### Q3: How do you access an object property?
**Answer:** object.property or object["property"]

### Q4: What does map() return?
**Answer:** A new array with transformed elements

### Q5: How do you check if an array includes a value?
**Answer:** array.includes(value)
