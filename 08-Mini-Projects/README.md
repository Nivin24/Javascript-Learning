# Mini Projects

## Overview

This folder contains four mini-projects to practice JavaScript skills:

1. **Calculator** - Basic arithmetic operations
2. **Todo App** - Task management with localStorage
3. **Weather App** - Fetch weather data from API
4. **Guess the Number** - Number guessing game

## How to Use

Each project has its own folder with:
- `index.html` - Structure
- `style.css` - Styling
- `script.js` - Logic

Open `index.html` in your browser to run each project.

## Learning Goals

- DOM manipulation
- Event handling
- Local storage
- API integration
- User interaction
- Form validation
