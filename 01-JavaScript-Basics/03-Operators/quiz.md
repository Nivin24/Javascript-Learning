# Operators - Quiz

### Question 1
What is the result of `10 % 3`?
- A) 3
- B) 1
- C) 0
- D) 3.33

**Answer:** B) 1 (remainder of 10 divided by 3)

### Question 2
What does `===` check?
- A) Value only
- B) Type only
- C) Both value and type
- D) Reference only

**Answer:** C) Both value and type

### Question 3
What is the output of `5 + 3 * 2`?
- A) 16
- B) 11
- C) 13
- D) 10

**Answer:** B) 11 (multiplication has higher precedence)

### Question 4
What does `++x` do?
- A) Increments x after returning value
- B) Increments x before returning value
- C) Decrements x
- D) Doubles x

**Answer:** B) Increments x before returning value

### Question 5
What is the result of `true && false`?
- A) true
- B) false
- C) undefined
- D) Error

**Answer:** B) false (AND requires both to be true)

### Question 6
What is `10 > 5 || 3 > 7`?
- A) true
- B) false
- C) undefined
- D) Error

**Answer:** A) true (OR requires at least one to be true)

### Question 7
What does the ternary operator `x ? y : z` mean?
- A) If x then y else z
- B) x equals y or z
- C) x is between y and z
- D) x plus y minus z

**Answer:** A) If x then y else z

### Question 8
What is `"5" + 3`?
- A) 8
- B) "8"
- C) "53"
- D) Error

**Answer:** C) "53" (string concatenation)

### Question 9
What is `!true`?
- A) true
- B) false
- C) undefined
- D) 1

**Answer:** B) false (NOT operator inverts)

### Question 10
What is `typeof (5 > 3)`?
- A) "number"
- B) "boolean"
- C) "string"
- D) "comparison"

**Answer:** B) "boolean" (comparison returns boolean)
