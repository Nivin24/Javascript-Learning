## Multiple Choice Questions

### Question 1
What is JavaScript primarily used for?
- A) Styling web pages
- B) Structuring web pages
- C) Adding interactivity to web pages
- D) Managing databases

**Answer:** C) Adding interactivity to web pages

---

### Question 2
Who created JavaScript?
- A) Tim Berners-Lee
- B) Brendan Eich
- C) Guido van Rossum
- D) James Gosling

**Answer:** B) Brendan Eich

---

### Question 3
What is ECMAScript?
- A) A JavaScript framework
- B) A JavaScript library
- C) The specification/standard for JavaScript
- D) A JavaScript compiler

**Answer:** C) The specification/standard for JavaScript

---

### Question 4
Which of the following can JavaScript NOT do?
- A) Manipulate HTML elements
- B) Validate form data
- C) Compile C++ code
- D) Fetch data from servers

**Answer:** C) Compile C++ code

What does "interpreted language" mean?

- B) Code runs directly without compilation

## True or False

### Question 9
JavaScript and Java are the same language.
- True
- False

**Answer:** False (They are completely different languages despite similar names)

---

### Question 10
JavaScript can only run in web browsers.
- True
- False

**Answer:** False (JavaScript can run on servers with Node.js, mobile apps, desktop apps, etc.)

## Short Answer Questions

### Question 11
Name three things you can build with JavaScript.

**Sample Answer:**
1. Interactive websites
2. Mobile applications (React Native)
3. Server-side applications (Node.js)

---

### Question 12
What is the difference between client-side and server-side JavaScript?

**Sample Answer:**
- **Client-side**: JavaScript runs in the user's browser to create interactive web pages
- **Server-side**: JavaScript runs on a server (using Node.js) to handle requests, process data, and interact with databases

---

### Question 13
Why is JavaScript called a "dynamically typed" language?

**Sample Answer:**
Variables in JavaScript don't have fixed types. The same variable can hold different types of data (number, string, object, etc.) at different times during program execution.
