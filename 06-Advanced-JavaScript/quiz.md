# Advanced JavaScript Quiz

### Q1: What is a closure?
**Answer:** A function that has access to variables from its outer function

### Q2: What are the three types of scope?
**Answer:** Global, function, and block scope

### Q3: What is hoisting?
**Answer:** Moving declarations to the top of their scope before execution

### Q4: What does `this` refer to in an object method?
**Answer:** The object that owns the method

### Q5: Do arrow functions have their own `this`?
**Answer:** No, they inherit `this` from the outer scope
