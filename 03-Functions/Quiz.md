# Functions Quiz

### Q1: What does a function return if no return statement is specified?
**Answer:** undefined

### Q2: What are default parameters?
**Answer:** Values assigned to parameters if no argument is provided

### Q3: What is the difference between function declaration and expression?
**Answer:** Declarations are hoisted, expressions are not

### Q4: What is an arrow function?
**Answer:** A shorter syntax for writing functions (ES6)

### Q5: Can arrow functions be used as methods?
**Answer:** Yes, but they don't have their own `this` binding
