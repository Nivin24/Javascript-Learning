# Loops - Quiz

### Question 1
What will this output?
```javascript
for (let i = 0; i < 3; i++) {
  console.log(i);
}
```
- A) 0 1 2
- B) 1 2 3
- C) 0 1 2 3
- D) 1 2

**Answer:** A) 0 1 2

### Question 2
What's the difference between while and do-while?
- A) No difference
- B) do-while executes at least once
- C) while is faster
- D) do-while is deprecated

**Answer:** B) do-while executes at least once

### Question 3
What does `break` do in a loop?
- A) Skips current iteration
- B) Exits the loop
- C) Restarts the loop
- D) Pauses the loop

**Answer:** B) Exits the loop

### Question 4
What does `continue` do?
- A) Exits the loop
- B) Skips to next iteration
- C) Restarts from beginning
- D) Stops the program

**Answer:** B) Skips to next iteration

### Question 5
How many times will this loop run?
```javascript
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
```
- A) 4
- B) 5
- C) 6
- D) Infinite

**Answer:** B) 5
