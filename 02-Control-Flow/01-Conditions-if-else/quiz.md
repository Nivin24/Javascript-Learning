# Conditional Statements - Quiz

### Question 1
What will this code output?
```javascript
let x = 10;
if (x > 5) {
  console.log("A");
} else {
  console.log("B");
}
```
- A) A
- B) B
- C) Nothing
- D) Error

**Answer:** A) A

### Question 2
Which value is falsy?
- A) "0"
- B) []
- C) 0
- D) "false"

**Answer:** C) 0

### Question 3
What is the output?
```javascript
let age = 18;
if (age >= 18) {
  console.log("Adult");
}
if (age < 18) {
  console.log("Minor");
}
```
- A) Adult
- B) Minor
- C) Both
- D) Nothing

**Answer:** A) Adult

### Question 4
What's the difference between `if-else if` and multiple `if` statements?
- A) No difference
- B) `if-else if` stops after first true condition
- C) Multiple `if` is faster
- D) `if-else if` is deprecated

**Answer:** B) `if-else if` stops after first true condition

### Question 5
What will this output?
```javascript
if ("") {
  console.log("True");
} else {
  console.log("False");
}
```
- A) True
- B) False
- C) Error
- D) Nothing

**Answer:** B) False (empty string is falsy)
